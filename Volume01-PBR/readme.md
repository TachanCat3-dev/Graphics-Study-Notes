# PBR基于物理的渲染学习知识总结



### Albedo 的最准确翻译是什么？

反照率（物理学/天文学标准翻译）： 这是字面和学术意义上最准确的翻译（源自拉丁语 albus，意为白色）。它指的是物体反射太阳辐射的比例。但这个词在游戏开发中显得过于学术和生僻，美术人员通常不太习惯这个称呼。

固有色（传统美术/早期 CG 翻译）： 这是一个容易引起误导的翻译。在传统绘画或非 PBR 渲染（如 Lambert/Blinn-Phong）中，“固有色”往往包含了画师或美术对物体阴影（AO）和微小高光的预烘焙。但在 PBR 规范中，Albedo 必须是完全去除光照和阴影信息的纯净颜色。因此，用“固有色”称呼它容易让美术习惯性地在贴图中画入阴影。

基础色 / Base Color（现代 PBR 引擎的最佳翻译）： 现代商业引擎（如 Unreal Engine、Substance Painter）干脆放弃了 Albedo 这个词，直接改叫 Base Color。这是目前最准确、最不易产生歧义的工程翻译。它清晰地表明：这只是一个基础的数据输入，它最终变成什么（漫反射还是高光），由 Metallic 参数来决定。

在学术讨论中叫“反照率”，但在实际工程和写 Shader 时，把它理解并翻译为“基础色 (Base Color)”是最合理的。


### 为什么非金属和金属可以被“金属度 (Metallic)”融合？

在真实的物理世界中，物质的导电性其实是泾渭分明的：一个材质要么是绝缘体（非金属），要么是导体（金属）。 在宏观渲染领域，根本不存在所谓的“半金属”（虽然物理上有半导体，但在游戏图形学中我们忽略不计）。

既然物理上非黑即白，那为什么引擎里会给你一个 0.0 到 1.0 的连续滑块（或一张灰度图）来融合二者呢？核心原因有两个：


A. 亚像素级别的材质混合（微观混合）
在游戏中，贴图的一个像素（Texel）往往对应着现实世界中一块不小的面积。假设一个金属表面生锈了，或者上面覆盖了一层灰尘：

生锈部分 = 纯非金属（Metallic = 0）
未生锈的金属部分 = 纯金属（Metallic = 1）

如果这两种材质在微观上交织在一起，由于分辨率的限制，引擎在一个像素内无法把它们分开渲染。此时，如果提供一个 Metallic = 0.5 的值，引擎其实是在表
达：“这个像素所覆盖的微观表面中，有 50% 是金属，50% 是非金属。”
这就实现了诸如“表面划痕”、“积灰的金属”、“磨损的烤漆”等复杂材质的平滑过渡。


B. 数学上的线性插值 (Lerp)
在 Shader 的底层逻辑中，引擎利用 Metallic 这个 0 到 1 的参数，巧妙地在两条物理规则之间做线性插值（Lerp）：

漫反射通道的融合：
Diffuse = BaseColor * (1.0 - Metallic)
当金属度从 0 滑向 1 时，原本明亮的漫反射颜色会被逐渐压暗，直到变成纯黑。

高光通道 (F0) 的融合：
F0 = lerp(float3(0.04, 0.04, 0.04), BaseColor, Metallic)
当金属度从 0 滑向 1 时，高光的反射率从一个无色的固定值 0.04，逐渐变成了你画在贴图上的那层带有色彩的 BaseColor。



### 绝缘体一般为单色/灰黑色镜面反射颜色 float表示   导体(金属)具有彩色镜面反射颜色float3表示

当光线击中非金属表面时，会发生两件事：一小部分光在表面直接弹开（镜面反射），其余大部分光穿透表面进入材质内部（折射，随后发生吸收和散射，形成漫反射/Diffuse）。折射率的均匀性： 决定表面镜面反射强度的物理量是折射率(Index of Refraction,IOR)

对于绝缘体而言，在可见光波段（约 400nm - 700nm）内，其折射率n随波长的变化非常微弱（虽然存在色散现象，但在PBR的宏观计算中完全可以忽略不计）。

反射均等 = 无色： 既然红、绿、蓝三种波长的光在表面的反射率几乎完全相同，那么反射出的光线就保留了光源原本的颜色。如果光源是白光，反射光就是白光。

为什么表现为灰色？ 因为非金属的表面基础反射率（f0）通常很低，大多数常见非金属（水、塑料、玻璃、木头）的 f0 都在2% ~ 5% 之间。白光被衰减到原来的 4%，在视觉上和数值上表现出来的就是暗灰色。

非金属的颜色从哪来？ 非金属的“颜色”完全来自于穿透表面的光线在材质内部经过吸收后，重新散射出来的光（漫反射 / Albedo）。


金属的微观结构与非金属截然不同，它内部充满了自由电子气（Sea of free electrons）。

强烈的光电相互作用： 光是一种电磁波。当光照射到金属表面时，光波的电场会强烈驱动金属表面的自由电子发生震荡。这种震荡会产生一个新的电磁波，将绝大部分光线直接在表面“弹”回去（这就是为什么金属几乎没有漫反射，光线无法穿透金属内部）。

吸收光谱的差异： 关键在于，不同金属的自由电子对不同波长（红、绿、蓝）的光束，其响应频率和吸收特性是不同的。金属的复折射率中包含一个很大的消光系数（Extinction Coefficient, $\kappa$）。

例如：金（Gold） 的电子结构会强烈吸收可见光光谱中的高频蓝光，而将低频的红光和绿光反射出去。红光 + 绿光 = 黄色，这就是为什么金子的镜面反射是金黄色的。

铜（Copper） 吸收了更多的蓝光和绿光，反射了红光，所以呈现红铜色。

反射带有色彩： 因为金属对RGB三原色的反射率各不相同，它的 $F_0$ 必须使用三个独立的值来分别记录红、绿、蓝通道的反射率。因此，金属的 $F_0$ 是一个 float3。
