# PBR基于物理的渲染学习知识总结

目前的游戏开发行业中PBR（Physically Based Rendering，基于物理的渲染）已经成为了主流的材质和渲染规范。它通过模拟光与物质的物理交互来实现更真实的视觉效果。
我认为学习理解PBR对于日后的图形学和游戏开发工作是至关重要的 PBR就像是新时代游戏开发的基础语法 掌握它就相当于掌握了现代游戏的命脉.
目前的很多二次元游戏也在向着PBR和NPR融合的方向发展 这方面很新的例子有:明日方舟终末地 __ __ 等作品

# Albedo 的最准确翻译是什么？

反照率（物理学/天文学标准翻译）： 这是字面和学术意义上最准确的翻译（源自拉丁语 albus，意为白色）。它指的是物体反射太阳辐射的比例。但这个词在游戏开发中显得过于学术和生僻，美术人员通常不太习惯这个称呼。

固有色（传统美术/早期 CG 翻译）： 这是一个容易引起误导的翻译。在传统绘画或非 PBR 渲染（如 Lambert/Blinn-Phong）中，“固有色”往往包含了画师或美术对物体阴影（AO）和微小高光的预烘焙。但在 PBR 规范中，Albedo 必须是完全去除光照和阴影信息的纯净颜色。因此，用“固有色”称呼它容易让美术习惯性地在贴图中画入阴影。

基础色 / Base Color（现代 PBR 引擎的最佳翻译）： 现代商业引擎（如 Unreal Engine、Substance Painter）干脆放弃了 Albedo 这个词，直接改叫 Base Color。这是目前最准确、最不易产生歧义的工程翻译。它清晰地表明：这只是一个基础的数据输入，它最终变成什么（漫反射还是高光），由 Metallic 参数来决定。

在学术讨论中叫“反照率”，但在实际工程和写 Shader 时，把它理解并翻译为“基础色 (Base Color)”是最合理的。


# 为什么非金属和金属可以被“金属度 (Metallic)”融合？

在真实的物理世界中，物质的导电性其实是泾渭分明的：一个材质要么是绝缘体（非金属），要么是导体（金属）。 在宏观渲染领域，根本不存在所谓的“半金属”（虽然物理上有半导体，但在游戏图形学中我们忽略不计）。

既然物理上非黑即白，那为什么引擎里会给你一个 0.0 到 1.0 的连续滑块（或一张灰度图）来融合二者呢？核心原因有两个：


A. 亚像素级别的材质混合（微观混合）
在游戏中，贴图的一个像素（Texel）往往对应着现实世界中一块不小的面积。假设一个金属表面生锈了，或者上面覆盖了一层灰尘：

生锈部分 = 纯非金属（Metallic = 0）
未生锈的金属部分 = 纯金属（Metallic = 1）

如果这两种材质在微观上交织在一起，由于分辨率的限制，引擎在一个像素内无法把它们分开渲染。
此时，如果提供一个 Metallic = 0.5 的值，引擎其实是在表达：“这个像素所覆盖的微观表面中，有 50% 是金属，50% 是非金属。”
这就实现了诸如“表面划痕”、“积灰的金属”、“磨损的烤漆”等复杂材质的平滑过渡。


B. 数学上的线性插值 (Lerp)
在 Shader 的底层逻辑中，引擎利用 Metallic 这个 0 到 1 的参数，巧妙地在两条物理规则之间做线性插值（Lerp）：

漫反射通道的融合：
Diffuse = BaseColor * (1.0 - Metallic)
当金属度从 0 滑向 1 时，原本明亮的漫反射颜色会被逐渐压暗，直到变成纯黑。

高光通道 (F0) 的融合：
F0 = lerp(float3(0.04, 0.04, 0.04), BaseColor, Metallic)
当金属度从 0 滑向 1 时，高光的反射率从一个无色的固定值 0.04，逐渐变成了你画在贴图上的那层带有色彩的 BaseColor。



# 绝缘体一般为单色/灰黑色镜面反射颜色 (float表示) 导体(金属)具有彩色镜面反射颜色float3表示

当光线击中非金属表面时，会发生两件事：一小部分光在表面直接弹开（镜面反射），其余大部分光穿透表面进入材质内部（折射，随后发生吸收和散射，形成漫反射/Diffuse）。折射率的均匀性： 决定表面镜面反射强度的物理量是折射率(Index of Refraction,IOR)

对于绝缘体而言，在可见光波段（约 400nm - 700nm）内，其折射率n随波长的变化非常微弱（虽然存在色散现象，但在PBR的宏观计算中完全可以忽略不计）。

反射均等 = 无色： 既然红、绿、蓝三种波长的光在表面的反射率几乎完全相同，那么反射出的光线就保留了光源原本的颜色。如果光源是白光，反射光就是白光。

为什么表现为灰色？ 因为非金属的表面基础反射率（f0）通常很低，大多数常见非金属（水、塑料、玻璃、木头）的 f0 都在2% ~ 5% 之间。白光被衰减到原来的 4%，在视觉上和数值上表现出来的就是暗灰色。

非金属的颜色从哪来？ 非金属的“颜色”完全来自于穿透表面的光线在材质内部经过吸收后，重新散射出来的光（漫反射 / Albedo）。


金属的微观结构与非金属截然不同，它内部充满了自由电子气（Sea of free electrons）。

强烈的光电相互作用： 光是一种电磁波。当光照射到金属表面时，光波的电场会强烈驱动金属表面的自由电子发生震荡。这种震荡会产生一个新的电磁波，将绝大部分光线直接在表面“弹”回去（这就是为什么金属几乎没有漫反射，光线无法穿透金属内部）。

吸收光谱的差异： 关键在于，不同金属的自由电子对不同波长（红、绿、蓝）的光束，其响应频率和吸收特性是不同的。金属的复折射率中包含一个很大的消光系数（Extinction Coefficient, $\kappa$）。

例如：金（Gold） 的电子结构会强烈吸收可见光光谱中的高频蓝光，而将低频的红光和绿光反射出去。红光 + 绿光 = 黄色，这就是为什么金子的镜面反射是金黄色的。

铜（Copper） 吸收了更多的蓝光和绿光，反射了红光，所以呈现红铜色。

反射带有色彩： 因为金属对RGB三原色的反射率各不相同，它的 $F_0$ 必须使用三个独立的值来分别记录红、绿、蓝通道的反射率。因此，金属的 $F_0$ 是一个 float3。

# 渲染方程

由 Kajiya 在 1986 年提出，是现代所有物理渲染算法的数学基础：

$$L_o(p, \omega_o) = L_e(p, \omega_o) + \int_{\Omega^+} L_i(p, \omega_i) f_r(p, \omega_i, \omega_o) (n \cdot \omega_i) d\omega_i$$

### 方程符号详解：

* **$L_o(p, \omega_o)$**：**出射辐射率 (Outgoing Radiance)**。表示最终从表面点 $p$ 沿着观察方向 $\omega_o$ （进入我们眼睛或相机）的光的总能量。


* **$L_e(p, \omega_o)$**：**自发光辐射率 (Emitted Radiance)**。表示点 $p$ 自身作为光源向 $\omega_o$ 方向直接发射的光能。如果物体本身不发光，这部分的值就是 0。


* **$\int_{\Omega^+}$**：**半球空间积分 (Integral over the hemisphere)**。表示我们要把点 $p$ 表面的整个法线上半球空间 ($\Omega^+$) 里的所有方向遍历一遍，把来自四面八方的光线能量收集累加起来。


* **$L_i(p, \omega_i)$**：**入射辐射率 (Incident Radiance)**。表示从外围环境的某个具体方向 $\omega_i$ 照射到点 $p$ 上的光能（这就是周围环境光或者其他光源打在物体上的光）。


* **$f_r(p, \omega_i, \omega_o)$**：**双向反射分布函数 (BRDF)**。这是渲染方程中体现“材质”的核心。它决定了从入射方向 $\omega_i$ 照射进来的光，有多少比例会被反射到观察方向 $\omega_o$ 去。不同材质（如金属、塑料、布料）的区别都由这个函数决定。


* **$(n \cdot \omega_i)$**：**余弦衰减/几何衰减项 (Cosine term)**。即表面法线 $n$ 与入射光方向 $\omega_i$ 的点积（等同于 $\cos\theta$）。根据兰伯特定律，光线照射表面的角度越倾斜，单位面积接收到的光能就越少，垂直照射时能量最大。


* **$d\omega_i$**：**立体角微分 (Differential solid angle)**。代表积分过程中，半球空间里极微小的一小束立体角区间。


**一句话总结渲染方程：**
**我们看到的某一点的光 ($L_o$) = 它自己发出的光 ($L_e$) + 对来自四面八方所有照射到这个点上的光 ($L_i$) 经过材质交互 ($f_r$) 和角度衰减 ($n \cdot \omega_i$) 后的反射能量总和。**

# 迪士尼的BRDF原则

目前正在手动写PBR_Library.hlsl 现在进度很安全 我会慢慢将我学到的知识总结在这里 同时用作项目书写的参考

迪士尼的BRDF原则是基于物理的渲染模型中一个非常重要的里程碑。它提出了一套简化但功能强大的参数化模型，使得艺术家能够更直观地控制材质的外观，同时保持物理上的合理性。
可以说对于现阶段的游戏和影视制作渲染已经称作是基础范式 所以理解迪士尼BRDF模型对于我们学习PBR是非常有帮助的

## 迪士尼BRDF模型中的主要参数：

baseColor（固有色）：表面颜色，通常由纹理贴图提供。

subsurface（次表面）：使用次表面近似控制漫反射形状。


metallic（金属度）：金属（0 = 电介质，1 =金属）。这是两种不同模型之间的线性混合。金属模型没有漫反射成分，并且还具有等于基础色的着色入射镜面反射。

specular（镜面反射强度）：入射镜面反射量。用于取代折射率。

specularTint（镜面反射颜色）：对美术控制的让步，用于对基础色（basecolor）的入射镜面反射进行颜色控制。掠射镜面反射仍然是非彩色的。

roughness（粗糙度）：表面粗糙度，控制漫反射和镜面反射。

anisotropic（各向异性强度）：各向异性程度。用于控制镜面反射高光的纵横比。（0 =各向同性，1 =最大各向异性。）

sheen（光泽度）：一种额外的掠射分量（grazing component），主要用于布料。

sheenTint（光泽颜色）：对sheen（光泽度）的颜色控制。

clearcoat（清漆强度）：有特殊用途的第二个镜面波瓣（specular lobe）。

clearcoatGloss（清漆光泽度）：控制透明涂层光泽度，0 = “缎面（satin）”外观，1 = “光泽（gloss）”外观。

## 核心BRDF模型

迪士尼采用了microfacet Cook-Torrance BRDF着色模型

### 核心公式
$$f(l, v) = \text{diffuse} + \frac{D(\theta_h) F(\theta_d) G(\theta_l, \theta_v)}{4 \cos\theta_l \cos\theta_v}$$

参数含义简述：

$f(l, v)$: 双向反射分布函数（BRDF），描述光线从入射方向$l$ 到观察方向$v$ 的反射规律

$diffuse$: 漫反射部分(Disney Diffuse)

$D(\theta_h)$: 法线分布函数（Normal Distribution Function），描述微平面法线与半角向量$h$ 一致的概率。

$F(\theta_d)$: 菲涅耳方程（Fresnel Equation），描述不同角度下的反射率。

$G(\theta_l, \theta_v)$: 几何遮蔽函数（Geometry Function），描述微平面相互遮挡导致的能量损耗。

$4 \cos\theta_l \cos\theta_v$: 校正因子，用于将微平面坐标系下的计算转换到宏观表面。
